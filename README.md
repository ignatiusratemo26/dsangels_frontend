# DSAngels - Empowering Girls in Tech 🌟

DSAngels is an interactive learning platform designed specifically for girls to explore technology, coding, and STEM fields in a fun, supportive, and engaging environment. Our platform makes learning technology accessible and magical for young learners.

## 🎯 Mission

To spark curiosity and build confidence in technology among girls by providing an inclusive, age-appropriate learning environment where every girl can discover the tech superhero within herself.

## ✨ Key Features

### 🎮 Interactive Learning
- **Fun Coding Challenges**: Solve exciting puzzles and games while learning to code
- **AI-Powered Assistant**: Personalized learning with magical AI guidance and hints
- **Visual Learning**: Interactive tutorials and concept visualizers for better understanding

### 🏆 Gamification & Progress
- **Achievement System**: Earn cool badges and rewards for completing challenges
- **Progress Tracking**: Visual progress bars and skill charts to track learning journey
- **Leaderboards**: Friendly competition to motivate continued learning

### 👥 Community & Mentorship
- **Mentor Connections**: Connect with industry professionals who guide and inspire
- **Role Models**: Discover inspiring women in technology and their stories
- **Discussion Forums**: Share achievements and learn together with peers
- **Community Chat**: Interactive discussions with AI chatbot support

### 📚 Educational Content
- **Age-Appropriate Curriculum**: Content designed specifically for different age groups
- **Concept Learning**: Interactive explanations of technology concepts
- **Skill Development**: Progressive learning paths in various tech domains

### 👤 User Profiles & Safety
- **Multiple Registration Types**: Students, Parents, and Mentors
- **Safe Environment**: Age-appropriate content with parental oversight options
- **Customizable Profiles**: Personal learning journeys and achievement showcases

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ignatiusratemo26/dsangels_frontend.git
   cd dsangels_frontend
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env
   # Edit .env file with your configuration
   ```

4. **Start development server**
   ```bash
   npm run dev
   ```

5. **Open your browser**
   Navigate to `http://localhost:5173` to see the application

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
